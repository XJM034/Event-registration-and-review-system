## 第一部分：快速开始

### 1. 系统简介

#### 1.1 这是什么系统？

这是一个体育赛事报名与审核管理系统，帮助赛事组织方高效管理报名流程，让教练便捷地为队伍报名，让队员轻松填写个人信息。

**系统主要功能：**
- 在线创建和管理赛事
- 自定义报名表单（根据不同赛事需求）
- 在线提交报名信息
- 在线审核报名
- 导出报名数据和附件
- 队员分享链接填写

#### 1.2 我是什么角色？

系统支持三种用户角色：

| 角色 | 权限说明 | 主要功能 |
|------|---------|---------|
| **超级管理员** | 完整系统管理权限 | 创建赛事、审核报名、管理账号、导出数据 |
| **普通管理员** | 赛事管理和审核权限 | 创建赛事、审核报名、导出数据 |
| **教练** | 报名和查看权限 | 浏览赛事、提交报名、查看审核结果 |
| **队员** | 填写个人信息 | 通过分享链接填写个人信息（无需登录） |

#### 1.3 如何登录？

**登录地址：** 系统登录页面（由管理员提供）

**登录方式：**
- 使用手机号作为账号
- 输入密码登录
- 管理员和教练使用同一个登录页面

**首次登录：**
- 管理员账号：由超级管理员创建并提供
- 教练账号：由超级管理员创建并提供
- 队员：无需登录，通过教练发送的分享链接直接填写

### 2. 角色快速指引

#### 2.1 管理员工作流程

```
创建赛事 → 配置报名表单 → 发布赛事 → 审核报名 → 导出数据
```

**典型操作流程：**
1. 登录系统
2. 创建新赛事，填写基本信息
3. 配置报名表单（设置需要收集的信息）
4. 设置报名时间和审核时间
5. 将赛事设为可见（教练端可见）
6. 等待教练提交报名
7. 审核报名信息（通过或驳回）
8. 导出已通过的报名数据

#### 2.2 教练工作流程

```
浏览赛事 → 填写报名 → 生成队员分享链接 → 提交报名 → 查看审核结果
```

**典型操作流程：**
1. 登录系统
2. 浏览可报名的赛事
3. 点击"立即报名"
4. 填写队伍信息
5. 添加队员信息（或生成分享链接让队员自己填）
6. 保存草稿（可随时修改）
7. 确认无误后提交报名
8. 在"我的通知"中查看审核结果
9. 如被驳回，修改后重新提交

#### 2.3 队员工作流程

```
收到分享链接 → 打开链接 → 填写个人信息 → 提交
```

**典型操作流程：**
1. 收到教练发送的分享链接
2. 在手机或电脑浏览器中打开链接
3. 填写个人信息（姓名、身份证号等）
4. 上传照片（如需要）
5. 点击提交
6. 查看提交成功提示

## 第二部分：管理员操作指南

### 3. 登录系统

#### 3.1 打开登录页面

1. 在浏览器中打开系统登录页面
2. 看到登录表单

#### 3.2 输入登录信息

1. 在"手机号"框中输入您的11位手机号
2. 在"密码"框中输入您的密码
3. 点击"登录"按钮

**提示：**
- 手机号必须是11位数字
- 密码区分大小写
- 如果多次输入错误，请联系超级管理员

#### 3.3 忘记密码怎么办？

如果忘记密码，请联系超级管理员重置密码。超级管理员可以在"账号管理"中为您重置密码。

### 4. 创建赛事

#### 4.1 进入创建页面

1. 登录后，在首页点击"创建赛事"按钮
2. 进入赛事创建表单

#### 4.2 填写基本信息

**必填信息：**

1. **赛事名称**
   - 输入完整的赛事名称
   - 例如："2026年全国青少年棍网球锦标赛"

2. **赛事类型**
   - 第一步：选择大类（棍网球、足球、篮球等）
   - 第二步：选择细分类型（男子/女子/混合）
   - 如果没有合适的选项，可以选择"其他"并自定义输入

4. **比赛时间**
   - 选择比赛开始日期
   - 选择比赛结束日期
   - 系统会自动检查：结束日期必须晚于或等于开始日期

5. **比赛地点**
   - 输入详细地址
   - 例如："北京市朝阳区奥林匹克公园体育场"

**可选信息：**

6. **赛事海报**
   - 点击上传区域选择图片
   - 支持格式：JPG、PNG、GIF、WebP
   - 文件大小：不超过5MB
   - 建议尺寸：800x600像素

7. **联系电话**
   - 输入赛事咨询电话

8. **赛事详情**
   - 输入赛事的详细介绍
   - 可以包含赛事背景、赛制说明等
   - 支持输入网址链接（系统会自动识别并转为可点击链接）

9. **报名要求**
   - 输入报名的特殊要求或注意事项

#### 4.3 保存赛事

1. 检查所有必填信息是否已填写
2. 点击"创建赛事"按钮
3. 看到成功提示后，赛事创建完成

**注意：**
- 新创建的赛事默认是"不可见"状态，教练端看不到
- 需要在赛事管理页面将其设为"可见"后，教练才能看到并报名

### 5. 配置报名表单

创建赛事后，需要配置报名表单，决定收集哪些信息。

#### 5.1 进入报名设置

1. 在赛事列表中，点击要配置的赛事
2. 在左侧菜单中点击"报名设置"
3. 进入报名设置页面

#### 5.2 设置报名时间

这是最重要的设置，决定了报名的时间范围。

**三个关键时间点：**

1. **报名开始时间**
   - 教练从这个时间开始可以提交报名
   - 在此之前，赛事显示"报名未开始"

2. **报名结束时间**
   - 教练必须在这个时间之前提交报名
   - 超过这个时间，无法新建或修改报名

3. **审核结束时间**
   - 管理员审核的截止时间
   - 超过这个时间，审核期结束

**时间设置规则：**
- 报名开始时间 < 报名结束时间
- 报名结束时间 < 审核结束时间
- 报名结束时间 < 比赛开始时间
- 审核结束时间 < 比赛开始时间

**示例时间线：**
```
4月1日 00:00  → 报名开始
4月15日 23:59 → 报名结束
4月20日 23:59 → 审核结束
4月25日 09:00 → 比赛开始
```

**提示：**
- 如果时间设置不合理，系统会显示红色提示
- 必须修正后才能保存

#### 5.3 配置队伍信息字段

决定收集哪些队伍信息。

**添加字段步骤：**

1. 在"队伍信息"区域，点击"添加字段"
2. 选择字段类型：
   - **文本**：单行文本输入（如队伍名称、联系人）
   - **图片**：上传单张图片（如队伍Logo）
   - **单选**：从多个选项中选一个（如参赛组别）
   - **多选**：可以选择多个选项
   - **日期**：选择日期
   - **单附件**：上传一个文档（PDF/Word/Excel）
   - **多附件**：上传多个文档

3. 填写字段设置：
   - **字段标签**：显示给用户的名称（如"队伍名称"）
   - **是否必填**：勾选后，教练必须填写此项
   - **选项配置**（单选/多选）：每行输入一个选项

4. 点击"确定"添加字段

**调整字段顺序：**
- 按住字段左侧的拖动图标
- 拖动到目标位置
- 松开鼠标完成排序

**提示：**
- 字段顺序会影响教练端报名表单的显示顺序和导出 Excel 的列顺序
- 管理端的审核列表和报名列表的显示列顺序是固定的，不受此处排序影响
- 建议把重要字段放在前面，这样教练填写时更顺手，管理员导出后也更方便查看和核对

#### 5.4 配置队员信息字段

决定收集哪些队员信息。

**添加角色：**

1. 点击"添加角色"按钮
2. 输入角色名称（如"运动员"、"教练"、"裁判"）
3. 点击"确定"

**为角色添加字段：**

1. 选择一个角色Tab
2. 点击"添加字段"
3. 选择字段类型（同队伍信息字段）
4. 填写字段设置
5. 点击"确定"

**常用队员字段：**
- 姓名（文本，必填）
- 性别（单选，必填）
- 出生日期（日期，必填）
- 身份证号码（文本，必填）
- 身份证照片（图片，必填）
- 联系电话（文本）

**特殊字段：身份证号码**
- 系统会自动验证身份证格式
- 自动提取性别、出生日期、年龄
- 自动验证是否符合赛事的年龄和性别要求

#### 5.5 组别与队员限制

队员的性别、年龄、人数等限制统一在"项目管理 → 组别设置"中维护，不在报名设置中单独配置。

**工作方式：**
1. 超级管理员在"项目管理"中创建组别，并为每个组别设置规则：
   - **性别限制**：不限 / 仅男子 / 仅女子 / 混合
   - **年龄限制**：通过出生日期范围限制（根据身份证号精确到天校验）
   - **人数限制**：最少/最多队员人数
2. 创建赛事时，赛事会关联到对应的项目和组别
3. 在报名设置页面，选择组别后会自动显示该组别的限制规则
4. 教练报名时，系统会根据组别规则自动校验队员信息

**提示：**
- 如果需要修改队员限制规则，请联系超级管理员在"项目管理"中调整
- 报名设置页面会显示当前组别的限制规则，但不能在此处修改

#### 5.6 上传导出模板

管理员可以上传报名表和运动员信息表的 PDF 模板。教练在报名时，系统会将填写的信息自动注入模板并导出。

**两种模板：**
- **报名表模板**：导出时注入队员、教练、领队等报名信息（1页）
- **运动员信息表模板**：导出时注入证件照、姓名和比赛服号码（2页）

**操作步骤：**

1. 点击"下载标准模板"获取官方 PDF 模板
2. 在标准模板基础上修改（如修改标题、附件编号等文案）
3. 点击"上传草稿 PDF"上传修改后的模板
4. 点击"预览"检查模板效果
5. 确认无误后，保存报名设置时模板会一并发布

**修改规范：**
- 请在标准模板基础上修改，不要从零制作
- 仅建议修改标题、附件编号等文案，不要移动表格线、照片框和字段位置
- 保持 A4 尺寸和原始页数（报名表1页，运动员信息表2页）
- 不要通过 Word/WPS 重排版后再导出

**提示：**
- 模板仅支持 PDF 格式
- 上传后可以随时替换草稿
- 模板上传后，教练端报名页面会出现"模板导出"功能

#### 5.7 保存设置

1. 检查所有设置是否正确
2. 点击页面右上角的"保存"按钮
3. 看到成功提示后，设置保存完成

**注意：**
- 保存前系统会检查时间设置是否合理
- 如果有错误，会显示红色提示，必须修正后才能保存

### 6. 审核报名

当教练提交报名后，管理员需要审核报名信息。

#### 6.1 查看待审核列表

1. 在赛事管理页面，点击左侧菜单中的"审核列表"
2. 列表中显示所有待审核的报名
3. 每条记录显示队伍摘要信息和提交时间

**提示：**
- 审核列表仅显示"待审核"状态的报名
- 左侧菜单上会显示待审核数量的角标

#### 6.2 打开审核详情

1. 在待审核列表中，点击某条报名右侧的"审核"按钮
2. 进入审核详情页面
3. 页面分为两部分：队伍信息和队员信息

#### 6.3 审核队伍信息

1. 查看队伍填写的所有信息
2. 如果信息无误，点击"无误"按钮
3. 如果信息有问题，点击"需修改"按钮，并填写备注说明问题

**图片和附件：**
- 图片字段可以点击放大查看
- 附件字段可以点击预览或下载

#### 6.4 审核队员信息

1. 查看每个队员的信息
2. 对每个队员分别标记"无误"或"需修改"
3. 如果标记"需修改"，填写备注说明问题

#### 6.5 通过或驳回

**通过报名：**
1. 检查完所有信息后，点击页面底部的"通过"按钮
2. 确认操作
3. 系统自动通知教练审核已通过

**驳回报名：**
1. 点击"驳回"按钮
2. 系统会根据您标记的"需修改"项自动生成驳回理由
3. 您也可以手动修改驳回理由
4. 确认驳回
5. 系统自动通知教练审核被驳回，并附上驳回理由

**提示：**
- 驳回后，教练可以在报名期内修改并重新提交
- 通过后，报名会出现在"报名列表"中

#### 6.6 管理已通过的报名

在"报名列表"Tab中可以查看所有已通过的报名：

1. 点击左侧菜单中的"报名列表"
2. 查看已通过审核的报名列表
3. 可以查看详情、导出数据
4. 如有需要，也可以将已通过的报名改为驳回

**手动添加报名：**
- 管理员可以在报名列表中手动添加一条已通过的报名
- 适用于线下报名等特殊情况

### 7. 导出报名数据

管理员可以将报名数据导出为 Excel 文件和附件压缩包。

#### 7.1 选择导出范围

**从审核列表导出：**
1. 在审核列表中，勾选要导出的报名
2. 点击"导出"按钮

**从报名列表导出：**
1. 在报名列表（已通过）中，勾选要导出的报名
2. 点击"导出"按钮

#### 7.2 下载导出文件

1. 点击导出后，系统开始生成文件
2. 生成完成后，浏览器自动下载一个 `.zip` 压缩包
3. 解压后可以看到：
   - 根据导出时选择的分组方式，可能先看到组别文件夹、单位文件夹，或直接看到队伍文件夹
   - 每支队伍一个 Excel 文件（`.xlsx`）
   - 附件文件夹（包含上传的图片和文档）

#### 7.3 导出文件结构

**Excel 文件内容：**
- 队伍信息 Sheet：包含队伍的所有字段
- 各角色 Sheet：每个角色（运动员/教练/裁判）一个独立的 Sheet
- 列顺序与报名设置中的字段顺序一致

**关于字段顺序：**
- 字段顺序会影响教练端报名表单的显示顺序和导出 Excel 的列顺序
- 管理端的审核列表和报名列表的显示列顺序是固定的，不受此处排序影响
- 建议把重要字段放在前面，这样教练填写时更顺手，管理员导出后也更方便查看和核对

**附件文件夹：**
- 按角色和字段名分文件夹组织
- 图片文件以队伍名称命名，方便识别

**提示：**
- 导出可能需要一些时间，请耐心等待
- 如果报名中包含大量图片，压缩包可能较大

### 8. 管理账号

超级管理员可以创建和管理教练账号和管理员账号。普通管理员只能管理自己的账号。

#### 8.1 进入账号管理

1. 点击页面顶部的"账号管理"菜单
2. 进入账号管理页面

**超级管理员看到三个Tab：**
- 教练账号：管理所有教练
- 管理员账号：管理所有管理员
- 我的账号：管理自己的账号

**普通管理员只看到：**
- 我的账号：管理自己的账号

#### 8.2 创建教练账号

1. 在"教练账号"Tab中，点击"创建账号"按钮
2. 填写信息：
   - **手机号**（必填）：11位手机号，作为登录账号
   - **默认密码**（必填）：至少6位，教练首次登录使用
   - **姓名**（可选）：教练姓名
   - **学校**（可选）：所属学校
   - **机构**（可选）：所属机构
3. 点击"创建"按钮
4. 创建成功后，将手机号和默认密码告知教练

#### 8.3 创建管理员账号

1. 在"管理员账号"Tab中，点击"创建账号"按钮
2. 填写信息：
   - **手机号**（必填）：11位手机号
   - **默认密码**（必填）：至少6位
   - **是否超级管理员**：勾选后拥有完整权限
3. 点击"创建"按钮

#### 8.4 重置密码

1. 在账号列表中找到目标账号
2. 点击"重置密码"按钮
3. 输入新密码（至少6位）
4. 点击"确认"
5. 将新密码告知用户

#### 8.5 启用/禁用账号

1. 在账号列表中找到目标账号
2. 点击"启用"或"禁用"按钮
3. 确认操作

**注意：**
- 禁用后，该用户无法登录系统
- 启用后，该用户恢复登录能力

#### 8.6 删除账号

1. 在账号列表中找到目标账号
2. 点击"删除"按钮
3. 确认操作

**删除保护机制：**
- 如果教练有正在进行的报名（待审核/已通过），系统会阻止删除
- 如果教练有未结束赛事的被驳回报名，系统也会阻止删除
- 草稿和已取消的报名会被自动清理
- 不能删除自己的账号
- 不能删除最后一个超级管理员

#### 8.7 管理自己的账号

所有管理员都可以在"我的账号"Tab中：
- 查看自己的账号信息
- 修改密码

## 第三部分：教练操作指南

### 9. 登录系统

#### 9.1 打开登录页面

1. 在浏览器中打开系统登录页面
2. 看到登录表单

#### 9.2 输入登录信息

1. 在"手机号"框中输入您的11位手机号
2. 在"密码"框中输入您的密码
3. 点击"登录"按钮
4. 登录成功后，自动跳转到教练端首页

**提示：**
- 首次登录请使用管理员提供的默认密码
- 建议登录后在"账号设置"中修改密码

### 10. 浏览赛事

#### 10.1 赛事列表

登录后，首页显示所有可报名的赛事。

**筛选赛事：**
- **Tab 筛选**：
  - 全部：显示所有赛事
  - 报名中：仅显示当前可以报名的赛事
  - 已结束：显示已结束的赛事
  - 我的报名：显示您已报名的赛事
- **类型筛选**：点击类型筛选按钮，勾选感兴趣的赛事类型

**赛事状态标识：**
- 🔵 **报名中**：当前可以报名
- 🟡 **审核中**：报名已截止，正在审核
- ⚪ **未开始**：报名尚未开始
- ⚫ **已结束**：赛事已结束

**操作按钮：**
- "去报名"：点击进入报名页面（报名中的赛事）
- "查看我的报名"：查看已提交的报名（已报名的赛事）
- "查看详情"：查看赛事详细信息

#### 10.2 查看赛事详情

1. 点击赛事卡片或"查看详情"按钮
2. 查看赛事的完整信息：
   - 赛事海报
   - 赛事名称、类型
   - 比赛时间和地点
   - 联系电话
   - 赛事详情（文字中的网址会自动变成可点击的链接）
   - 报名要求
   - 附件下载

**报名状态区域：**
- 如果您已报名，页面会显示当前报名状态
- 根据状态显示不同的操作按钮

### 11. 填写报名

#### 11.1 进入报名页面

1. 在赛事列表中，点击"去报名"按钮
2. 或在赛事详情页中，点击"立即报名"按钮
3. 进入报名表单页面

#### 11.2 填写队伍信息

1. 默认显示"队伍信息"Tab
2. 按照表单要求填写各项信息
3. 带红色星号（*）的字段为必填项

**不同字段类型的填写方式：**
- **文本**：直接在输入框中输入
- **图片**：点击上传区域，选择图片文件
- **单选**：点击下拉菜单，选择一个选项
- **多选**：勾选一个或多个选项
- **日期**：点击日期选择器，选择日期
- **附件**：点击上传区域，选择文件

#### 11.3 填写队员信息

1. 切换到"队员信息"Tab
2. 如果有多个角色（运动员、教练、裁判），会显示角色子Tab
3. 点击"添加队员"按钮添加新队员
4. 填写每个队员的信息

**身份证号码填写：**
- 输入18位身份证号码
- 系统会实时验证格式是否正确
- 验证通过后，系统自动提取性别和出生日期
- 如果性别不符合赛事组别的性别要求，会显示提示
- 如果年龄不符合赛事组别的年龄要求，会显示提示

**删除队员：**
- 点击队员卡片右上角的删除按钮
- 确认删除

**人数提示：**
- 如果赛事设置了人数限制，页面会显示当前人数和要求
- 例如："当前 8 人，要求 10-20 人"

#### 11.4 保存草稿

1. 填写过程中，随时可以点击"保存草稿"按钮
2. 草稿保存后，下次可以继续编辑
3. 草稿不会进入审核流程

**提示：**
- 建议填写过程中定期保存草稿，避免信息丢失
- 草稿状态下可以随时修改任何信息

#### 11.5 导出报名表和运动员信息表

如果管理员已上传模板，报名页面会显示"模板导出"区域，可以将当前填写的信息导出为标准化的 PDF 文件。

**前提条件：**
- 报名记录已保存为草稿或已提交
- 管理员已为该赛事上传了对应的模板

**可导出的文件：**
- **报名表**：包含参赛单位、队伍、组别、领队/教练信息和队员报名信息
- **运动员信息表**：包含领队、教练、队员证件照、姓名和比赛服号码

**操作步骤：**
1. 在报名页面底部找到"模板导出"区域
2. 选择要导出的类型（报名表 / 运动员信息表）
3. 点击"下载 PDF"按钮
4. 系统将当前填写的信息注入模板并生成 PDF
5. 浏览器自动下载生成的文件

**提示：**
- 如果部分字段尚未填写，导出文件中对应位置会留空
- 建议在信息填写完整后再导出
- 如果看不到"模板导出"区域，说明管理员尚未上传模板

#### 11.6 提交报名

1. 确认所有信息填写完整
2. 点击"提交报名"按钮
3. 系统会检查所有必填项是否已填写
4. 如果有遗漏，会显示提示
5. 确认无误后，报名提交成功
6. 报名状态变为"待审核"

**注意：**
- 提交后，报名进入审核队列，等待管理员审核
- 提交后无法直接修改，需要等待审核结果
- 如果被驳回，可以修改后重新提交

### 12. 使用队员分享链接

如果队员人数较多，可以生成分享链接让队员自己填写个人信息。

#### 12.1 生成分享链接

1. 在报名页面的"队员信息"Tab中
2. 先添加一个队员（至少需要一个队员位置）
3. 点击队员卡片上的"生成分享链接"按钮
4. 系统生成一个专属链接

#### 12.2 发送链接给队员

1. 点击"复制链接"按钮
2. 将链接通过微信、短信等方式发送给队员
3. 每个队员有独立的链接，不要混淆

#### 12.3 查看填写状态

- 队员填写完成后，队员卡片上会显示"已填写"标识
- 您可以查看队员填写的信息

**注意事项：**
- 分享链接有效期为7天
- 链接过期后需要重新生成
- 报名必须处于"草稿"或"已驳回"状态才能使用分享链接
- 队员填写的信息会自动同步到您的报名表中

### 13. 查看报名状态

#### 13.1 打开"我的报名"

1. 点击左侧菜单中的"我的报名"
2. 查看所有已提交的报名

**筛选报名：**
- 全部：显示所有报名
- 待审核：显示等待审核的报名
- 已通过：显示审核通过的报名
- 已驳回：显示被驳回的报名

#### 13.2 报名状态说明

| 状态 | 含义 | 可执行操作 |
|------|------|-----------|
| **草稿** | 还未提交，可以继续编辑 | 编辑、提交、删除 |
| **待审核** | 已提交，等待管理员审核 | 查看、取消 |
| **已通过** | 审核通过 | 查看、取消 |
| **已驳回** | 审核未通过，需要修改 | 查看驳回理由、修改、重新提交 |
| **已取消** | 已取消报名 | 查看 |

#### 13.3 处理驳回

如果报名被驳回：

1. 在"我的报名"中找到被驳回的报名
2. 查看驳回理由（显示在报名卡片上）
3. 点击"重新报名"按钮
4. 根据驳回理由修改相关信息
5. 修改完成后重新提交

**提示：**
- 只有在报名期内才能重新提交
- 如果报名期已过，无法重新提交

### 14. 取消报名

#### 14.1 什么时候可以取消？

以下状态的报名可以取消：
- 草稿
- 待审核
- 已通过

#### 14.2 取消后能否重新报名？

取消后能否重新报名取决于当前所处的阶段：

| 报名状态 | 报名期内取消 | 审核期内取消 |
|---------|------------|------------|
| 草稿 | ✅ 可以重新提交 | ✅ 可以重新提交 |
| 待审核 | ✅ 可以重新提交 | ❌ 无法重新提交 |
| 已通过 | ✅ 可以重新提交 | ❌ 无法重新提交 |

#### 14.3 如何取消报名

1. 在"我的报名"或赛事详情页中
2. 点击"取消报名"按钮
3. 系统会显示确认对话框，提示取消后的影响
4. 确认取消

**注意：**
- 取消操作不可撤销
- 请仔细阅读确认对话框中的提示

### 15. 查看通知

系统会在报名审核完成后自动发送通知。

#### 15.1 通知入口

**通知类型：**
- **审核通过**：您的报名已通过审核
- **审核驳回**：您的报名被驳回，附有驳回理由
- **提醒**：其他系统提醒

**通知内容：**
- 赛事名称
- 队伍信息预览
- 通知详情
- 通知时间

#### 15.3 通知操作

- **标记已读**：点击单条通知的"标记已读"按钮

## 第五部分：常见问题

### 19. 登录问题

**Q：忘记密码怎么办？**

A：请联系超级管理员，管理员可以在"账号管理"中为您重置密码。

**Q：账号被禁用怎么办？**

A：请联系超级管理员，管理员可以在"账号管理"中重新启用您的账号。

**Q：输入手机号提示格式错误？**

A：请确保输入的是11位数字的手机号，不要包含空格、横线等字符。

**Q：登录后页面空白或报错？**

A：请尝试以下操作：
1. 刷新页面
2. 清除浏览器缓存
3. 更换浏览器（推荐使用 Chrome）
4. 如果问题持续，联系管理员

### 20. 报名问题

**Q：报名时间过了还能报名吗？**

A：不能。报名结束时间过后，无法新建或提交报名。请关注赛事的报名时间。

**Q：草稿会自动保存吗？**

A：不会自动保存。请在填写过程中定期点击"保存草稿"按钮。

**Q：提交后还能修改吗？**

A：提交后无法直接修改。需要等待管理员审核：
- 如果被驳回，可以修改后重新提交
- 如果已通过，可以取消报名后重新报名（仅在报名期内）

**Q：队员分享链接过期了怎么办？**

A：教练可以在报名页面重新为该队员生成新的分享链接。新链接有效期为7天。

**Q：可以同时报名多个赛事吗？**

A：可以。每个赛事的报名是独立的。

**Q：报名信息填错了怎么办？**

A：
- 如果是草稿状态：直接修改即可
- 如果已提交：等待审核结果，被驳回后可以修改
- 如果已通过：可以取消报名后重新报名（仅在报名期内）

### 21. 审核问题

**Q：审核需要多长时间？**

A：审核时间取决于管理员的工作安排，通常在报名结束后集中审核。请关注"我的通知"获取审核结果。

**Q：驳回后如何重新提交？**

A：
1. 在"我的报名"中找到被驳回的报名
2. 查看驳回理由
3. 点击"重新报名"
4. 根据驳回理由修改信息
5. 重新提交

**Q：审核期内可以取消报名吗？**

A：可以取消，但审核期内取消后无法重新提交。请谨慎操作。

### 22. 文件上传问题

**Q：支持哪些图片格式？**

A：支持 JPG、PNG、GIF、WebP 格式。

**Q：文件大小限制是多少？**

A：
- 图片：单个文件不超过 5MB
- 附件（PDF/Word/Excel）：单个文件不超过 10MB

**Q：上传失败怎么办？**

A：请检查以下几点：
1. 文件格式是否正确
2. 文件大小是否超过限制
3. 网络连接是否正常
4. 如果问题持续，尝试更换浏览器或刷新页面

### 23. 身份证验证问题

**Q：为什么提示身份证格式错误？**

A：身份证号码必须满足以下条件：
- 共18位
- 前17位必须是数字
- 第18位可以是数字或字母X（大小写均可）
- 校验位必须正确

请仔细核对身份证号码是否输入正确。

**Q：年龄不符合要求怎么办？**

A：如果系统提示年龄不符合赛事要求，说明该队员的年龄超出了赛事规定的范围。请确认：
1. 身份证号码是否输入正确
2. 该赛事的年龄要求是什么
3. 如有疑问，请联系赛事管理员

## 第六部分：附录

### 24. 报名状态说明

| 状态 | 图标颜色 | 含义 |
|------|---------|------|
| 草稿 | 灰色 | 报名信息还未提交，可以随时编辑 |
| 待审核 | 蓝色 | 报名已提交，等待管理员审核 |
| 已通过 | 绿色 | 管理员已审核通过 |
| 已驳回 | 红色 | 管理员审核未通过，需要修改后重新提交 |
| 已取消 | 灰色 | 报名已被取消 |

### 25. 报名阶段说明

一个赛事的报名流程分为以下阶段：

```
报名未开始 → 报名中 → 审核中 → 已结束
```

| 阶段 | 含义 | 教练可以做什么 |
|------|------|--------------|
| 未开始 | 报名开始时间还没到 | 查看赛事信息，等待报名开始 |
| 报名中 | 当前可以报名 | 创建报名、编辑草稿、提交报名、取消报名 |
| 审核中 | 报名已截止，管理员正在审核 | 查看报名状态、取消报名（不可重新提交） |
| 已结束 | 审核已结束 | 查看最终结果 |

### 26. 联系方式

如果在使用过程中遇到问题，请联系：
- 赛事相关问题：联系赛事管理员（赛事详情页中有联系电话）
- 账号问题：联系超级管理员
- 系统故障：联系技术支持
